# Fabric vs BTGenBot comparison

This repository contains the experiments designed and executed to evaluate the properties of the Fabric.

## **Experiment 1** 
Experiment 1 was designed to compare the generative properties of the Fabric against BTGenBot, and was developed for the ICRA2026 paper. All the source code required to test the system are available withn the `Experiment_1` folder with instructions

## **Experiment 2** 

Experiment 2 was designed to orchestrate complex behaviour planning using Fabric and was developed for the IROS2026 paper based on ICRA2026 feedback. All the source code required to test the system are available withn the `Experiment_2` folder with instructions. 

Please note that the software used in the Experiment 2 is newer compared to the Experiment 1, resulting in minor changes between syntax between two experiments.
